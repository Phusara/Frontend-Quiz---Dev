# Dek-D_Quiz

โปรเจคนี้ทำขึ้นมาสำหรับ Quiz Frontend 

# Tool
- Vue 3
- VScode
- Tailwind
- Daisy UI


## Project Run 
วิธีการ Run project ทำตามดังนี้
สร้างไฟล์ในไฟล์ .env ใน path ที่ clone ลงมาแล้วพิมพ์
```
VITE_APP_URL=http://localhost:5000
```
จากนั้นไปที่ cmd ใน VScode แล้วใส่ command
```cmd
npm install
```

```cmd
npm run backend
```

```cmd
npm run dev
```
